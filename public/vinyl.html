🎵 How to connect your Discogs collection:

  1. Make sure your Discogs collection is set to public
  2. Get your Discogs username from your profile URL
  3. Update the username in the vinyl.js file
  4. Optional: Get a Discogs API token for higher rate limits

Once configured, your entire collection will be displayed here with cover art, release information, and estimated values!
